# RestCoreAPI

API RESTful feita em C# com ASP.NET Core para gerenciamento de Produtos e Usuários.

## Tecnologias

- ASP.NET Core
- Entity Framework Core
- SQL Server (LocalDB)
- Swagger para documentação

## Endpoints

- `/produtos` - GET, POST, PUT, DELETE
- `/usuarios` - GET, POST, PUT, DELETE
